useThrottle 节流 ​

useThrottle 是 Vue 的 Hooks，用于节流。

基础用法 ​

tsx
import { useThrottle } from '@/hooks/useThrottle'

const throttle = useThrottle(fn, delay)

使用场景 ​

  • 滚动事件节流
  • 窗口大小改变事件节流

TIP

useThrottle 是参考 lodash 的 throttle 函数实现的。 具体用法可以参考 lodash。

API ​

参数说明类型默认值
fn需要节流的函数(...args: any[]) => void-
delay节流时间number1000
